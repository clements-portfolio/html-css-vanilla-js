Frequently Asked Questions (FAQ)

What is the Symbio API?

The Symbio API was developed to provide a higher level of service to clients of Clementine Solutions. We use the API to improve your development process, manage and store data, process site requests, retrieve data from other website, and much more.

What benefits does the Symbio API provide?

With Symbio, Clients are able to store data without worrying about additional database costs, use different types of software and devices with the same database, grow very quickly without the concern of running out of storage capacity, more easily and readily view and update the content and resources on websites and applications, and meet all privacy and security standards and legal requirements for these technologies.

The Symbio API is written in PHP, maintained on Heroku, and leverages MongoDB Atlas for cloud-based data storage. All data transmission and storage methods meet PCI DSS security standards and data storage policies comply with the California Privacy Protection Act (CPPA). User and client privacy and security is our top priority.

Do I have to use the Symbio API if I develop with Clementine Solutions?

No. Clients are free to choose whichever services they wish when it comes to their websites and technologies. Clementine Solutions may charge additional development costs to clients who require a more customized back end or data flow.

How much does the Symbio API cost?

The base price for a Symbio API license is $34.99 monthly or $364.99 annually per account per company. The base price covers up to 25,000 API requests per month after which clients are billed at the standard rate of $0.007 (seven tenths of 1 cent) per request thereafter until the next billing cycle.
